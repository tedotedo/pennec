# PENNEC website

Static site for the **Paediatric Epilepsy Network — North East & Cumbria** (PENNEC).

## Files
- `index.html` — single-page site with Home, About, Meetings, Guidelines, Publications, Videos, Local Support, Contact.
- `assets/styles.css` — design system (Fraunces + Inter, navy/amber/cream palette).
- `assets/script.js` — mobile menu, footer year, graceful image fallback.
- `assets/pennec-branding.jpg` — **add your branding photo here** (the child-crafting image). If absent, the hero shows a coloured gradient backdrop.

## Local preview
Open `index.html` in any browser — no build step required.

## Deploy
Drop the folder into Netlify / GitHub Pages / any static host. The target repo is https://github.com/tedotedo/pennec.
